# ✅ 部署前检查清单

在将应用部署到生产环境之前，请逐项检查以下内容。

---

## 📋 环境准备

### 服务器要求
- [ ] CPU: 2核心以上
- [ ] 内存: 4GB以上（推荐8GB）
- [ ] 磁盘: 20GB以上可用空间
- [ ] 操作系统: Linux（推荐Ubuntu 20.04+）

### 软件安装
- [ ] Python 3.8+ 已安装
- [ ] Node.js 16+ 已安装
- [ ] Nginx 已安装（如使用）
- [ ] 数据库已安装（PostgreSQL/MySQL，如不使用SQLite）

---

## 🔐 安全配置

### 密钥和密码
- [ ] 修改了默认管理员密码（admin/admin123）
- [ ] 生成了新的 `SECRET_KEY`（至少32位随机字符）
- [ ] Google API Key 已配置
- [ ] OSS密钥已配置（如使用）
- [ ] 所有密钥存储在 `.env` 文件中，未提交到Git

### 权限设置
- [ ] 后端上传目录有正确的写权限
- [ ] 数据库文件有正确的权限（如使用SQLite）
- [ ] 日志目录有写权限
- [ ] 不要以root用户运行应用

---

## 🗄️ 数据库配置

### 数据库选择
- [ ] 生产环境使用 PostgreSQL 或 MySQL（不推荐SQLite）
- [ ] 数据库已创建
- [ ] 数据库用户已创建并授权
- [ ] `DATABASE_URL` 正确配置

### 数据库初始化
- [ ] 已运行 `python init_database.py`
- [ ] 管理员账户创建成功
- [ ] 初始邀请码已生成
- [ ] 数据库表结构正确

### 备份策略
- [ ] 配置了自动备份脚本
- [ ] 测试了备份恢复流程
- [ ] 备份文件存储在安全位置

---

## 🌐 网络和域名

### 域名配置
- [ ] 域名已注册
- [ ] DNS记录已配置（A记录指向服务器IP）
- [ ] SSL证书已申请（Let's Encrypt或其他）
- [ ] HTTPS已配置

### 防火墙设置
- [ ] 开放80端口（HTTP）
- [ ] 开放443端口（HTTPS）
- [ ] 限制5000端口（后端API仅内网访问）
- [ ] 配置了必要的安全组规则

### CDN和加速
- [ ] 静态资源已配置CDN（可选）
- [ ] OSS已启用CDN加速（可选）

## ⚙️ 配置文件检查

### 后端配置 (.env)
```bash
# 必需配置

- [ ] DEBUG=False

# 数据库配置

# OSS配置（如使用）
- [ ] OSS_ACCESS_ID 已设置
- [ ] OSS_ACCESS_KEY 已设置
- [ ] OSS_ENDPOINT 正确
- [ ] OSS_BUCKET 已创建
- [ ] OSS_ENABLED=true

# 其他配置
- [ ] MAX_CONTENT_LENGTH 合理（默认50MB）
- [ ] API_TIMEOUT 合理（默认120秒）
```


### Nginx配置

