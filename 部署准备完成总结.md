## 🚀 部署步骤概览

### 快速部署（开发环境）
```bash
# 1. 后端
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env      # 编辑并填入API密钥
python init_database.py
python start_server.py

# 2. 前端
npm install
npm run dev
```

### 生产环境部署
1. 阅读 `DEPLOYMENT.md`
2. 检查 `PRE_DEPLOY_CHECKLIST.md`
3. 配置Nginx + Gunicorn或使用Docker
4. 申请HTTPS证书
5. 配置监控和日志
6. 执行部署
7. 验证功能
