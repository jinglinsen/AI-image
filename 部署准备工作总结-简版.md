## 🚀 快速部署（3步）

### 步骤1: 后端部署
```bash
cd backend
python -m venv venv
source venv/bin/activate         # Windows: venv\Scripts\activate
pip install -r requirements.txt
python init_database.py
python start_server.py
```

### 步骤2: 前端部署
```bash
npm install
npm run dev                      # 开发环境
# 或
npm run build                    # 生产环境
```

### 步骤3: 访问应用
- 前端: http://localhost:3000
- 后端: http://localhost:5000
- 管理员: `admin` / `admin123`
